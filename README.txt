TEAM TASK MANAGER

OVERVIEW
Team Task Manager is a full-stack web application that helps teams manage projects and tasks efficiently using role-based access control.

FEATURES

* User Authentication (Signup/Login)
* Role-Based Access (Admin & Member)
* Project Management
* Task Creation and Assignment
* Task Status Tracking
* Overdue Task Warnings
* Responsive Dashboard

TECH STACK

Frontend:

* React.js
* CSS3
* Axios

Backend:

* Node.js
* Express.js

Database:

* MongoDB Atlas

Deployment:

* Railway
* Vercel

USER ROLES

Admin:

* Create Projects
* Create Tasks
* Assign Tasks
* Delete Tasks
* Update Task Status

Member:

* View Tasks
* Update Task Status
